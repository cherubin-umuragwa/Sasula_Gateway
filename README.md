Getting Started

1) Install dependencies
```bash
npm install
```

2) Configure environment (.env)
```bash
cp .env.example .env
# fill PRIVATE_KEY, BASE_SEPOLIA_RPC_URL, BASESCAN_API_KEY
```

3) Run tests
```bash
npm run test
```

4) Deploy to Base Sepolia
```bash
npm run deploy:base
```
